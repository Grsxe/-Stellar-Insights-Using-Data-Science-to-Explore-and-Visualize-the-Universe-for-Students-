# 🌌 Stellar Insights

## Overview
This project uses data science techniques to explore and visualize star data, making astronomy concepts easier to understand for students.

## Features
- Data cleaning and preprocessing
- Visualization of star temperature and luminosity
- Analysis of spectral classes
- Insights into astronomical patterns

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insight
Hotter stars tend to be more luminous, and spectral classification helps group stars based on temperature.
